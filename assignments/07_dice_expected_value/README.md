# Assignment 07 — Dice Rolling: PDF and Expected Value

## Problem
Roll a die up to 3 times, stopping when an even number appears.
Record the sum of all rolls. Compute PDF and expected values.

## Random Variables
| Variable | Description                         |
|----------|-------------------------------------|
| Y        | Number of rolls needed (1, 2, or 3) |
| X        | Sum of all recorded rolls           |

## Results
| Variable | Expected Value |
|----------|----------------|
| E(Y)     | 1.84           |
| E(X)     | 6.53           |
